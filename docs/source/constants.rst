*********
Constants
*********

.. currentmodule:: pyquickbench

Pyquickbench includes several constants:

.. data:: default_ax_name
   
   The default value for benchmarked functions arguments.
   
   ``pyquickbench.default_ax_name = 'n'``   
.. data:: fun_ax_name
   
   The axis name for functions to be benchmarked.
   
   ``pyquickbench.fun_ax_name = 'function'``   
.. data:: repeat_ax_name
   
   The axis name for reapeated benchmarks.
   
   ``pyquickbench.repeat_ax_name = 'repeat'``
.. data:: out_ax_name
   
   The axis name for function output.
   
   ``pyquickbench.out_ax_name = 'out'``
